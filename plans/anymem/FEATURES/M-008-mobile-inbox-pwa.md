# M-008 - Mobile Inbox PWA

## Status
- Priority: `P1`
- Phase: `3 - Standalone Surfaces`
- State: `Planned`

## Goal
Provide mobile approval inbox with same-screen agent chat and non-binary dispositions.

## Acceptance Criteria
- End-to-end approval loop is usable without desktop.
- Reason, context, trace/proof links, and chat are available in one flow.

## Dependencies
- `M-005`, `M-006`, `M-010`
